Use it in Floe

This sample library is compatible with Floe, a free and open-source sampler plugin.

The file called __LUA_FILENAME__ instructs Floe how to load the samples in this folder. Therefore, to avoid problems in Floe, do not rearrange, rename or delete the files in this folder.

However, the library as-a-whole is portable: you can move it or back it up elsewhere.

Floe can be downloaded from here. There's also an online User Manual available, which includes information about how to make your own Floe libraries.

Use it outside Floe

You may use the samples in this library outside of Floe: they're just audio files.
