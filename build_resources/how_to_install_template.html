How to install

Option 1: use Floe

Use Floe's 'install library' button and direct it to the name.floe.zip file. It will extract the zip to the correct location.

Option 2: manual install

  1. Unzip the name.floe.zip file.
  2. Open the Library folder and put it's contents in /path/to/Floe/Libraries/.
  3. Open the Presets folder and put it's contents in /path/to/Floe/Presets/.

Instructions can also be found online at Floe User Manual.
